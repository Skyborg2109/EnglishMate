Conditional Sentences (Kalimat Pengandaian) adalah kalimat yang menyatakan kondisi dan hasil dari kondisi tersebut. Terdiri dari dua bagian: if clause (syarat) dan main clause (hasil).

1. Type 0 - General Truth (Kebenaran Umum)

Digunakan untuk menyatakan fakta umum atau kebiasaan yang selalu benar.

If + Simple Present, Simple Present

  • If you heat water to 100Â°C, it boils.
  • If it rains, the ground gets wet.
  • If you mix blue and yellow, you get green.

2. Type 1 - Real Possibility (Kemungkinan Nyata)

Digunakan untuk situasi yang mungkin terjadi di masa depan.

If + Simple Present, will + Verb 1

  • If it rains tomorrow, I will stay at home.
  • If you study hard, you will pass the exam.
  • If she comes, we will start the meeting.

3. Type 2 - Unreal Present (Tidak Nyata Sekarang)

Digunakan untuk situasi yang tidak mungkin atau bertentangan dengan kenyataan sekarang.

If + Simple Past, would + Verb 1

  • If I had a car, I would drive to work. (Kenyataan: Saya tidak punya mobil)
  • If she were rich, she would travel the world. (Kenyataan: Dia tidak kaya)
  • If I knew his number, I would call him. (Kenyataan: Saya tidak tahu nomornya)

4. Type 3 - Unreal Past (Tidak Nyata Masa Lalu)

Digunakan untuk situasi yang tidak terjadi di masa lalu (penyesalan).

If + Past Perfect, would have + Verb 3

  • If I had studied, I would have passed the exam. (Kenyataan: Saya tidak belajar dan tidak lulus)
  • If she had known, she would have come. (Kenyataan: Dia tidak tahu dan tidak datang)
  • If they had left earlier, they would have arrived on time. (Kenyataan: Mereka tidak berangkat lebih awal)
